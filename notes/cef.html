Conditional expectation function (CEF)

In a first course on probability, we learn about Bayes' rule: for any two events $A$ and $B$, we have \begin{align*} \mathbb{P}(A\cap B) = \mathbb{P}(A\mid B)\mathbb{P}(B). \end{align*} In this note, we will revisit this from the machinery given by conditional expectation.

At some point, we learn that conditional expectation is a random variable. In particular, if $X$ and $Y$ are random variables over $(\Omega, \mathcal{F},\mathcal{P})$, then the conditional expectation $E[Y\mid X]$ is a random variable that is $X$-measureable and satisfies an integration property.
